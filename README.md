## Tecnologias
   - Node
   - Sequelize ORM
   - Express
   - MYSQL

## Como executar
##### Clone o repositório
```bash
git clone https://github.com/alissonrsantos/sign-up-api.git
```
##### Abra a pasta
```bash
cd sign-up-api
```

##### Abra no Vscode
```bash
code .
```

##### Instale as dependências
```bash
npm install
```
ou

```bash
yarn
```

##### Abra no navegador
```bash
npm run dev
```
ou

```bash
yarn dev
```
